# Generic-Mini-Projects

A collection of self-contained frontend mini-projects built for learning, exploration, and developer tooling. Each project ships with its own stack, design system, and documentation.

---

## Projects

| # | App Name | Project Folder | Description | Hosted |
|---|----------|---------------|-------------|--------|
| 1 | **GasGlow** | [GasGlow\_Web3-Gas-Price-Tracker](https://github.com/Lakshit-Chiranjiv/Generic-Mini-Projects/tree/main/GasGlow_Web3-Gas-Price-Tracker) | Real-time Ethereum gas price tracker with Slow/Standard/Fast tiers, interactive 24h trend chart, custom threshold alerts, desktop push notifications, and a triple-stream data engine (mock / public RPC / Etherscan). | — |
| 2 | **CronOps** | [CronOps\_Crontab-Visualizer](https://github.com/Lakshit-Chiranjiv/Generic-Mini-Projects/tree/main/CronOps_Crontab-Visualizer) | Crontab expression parser and visualizer that translates any 5-field cron string into plain English, highlights the active field under the cursor, and shows the next 10 execution times with timezone support. | — |
| 3 | **NetOps** | [NetOps\_IP-Network-Info-Tool](https://github.com/Lakshit-Chiranjiv/Generic-Mini-Projects/tree/main/NetOps_IP-Network-Info-Tool) | IP and network information dashboard built with Next.js — shows geolocation, ISP details, DNS lookup, and port checker for any IP or domain. | — |
| 4 | **PingProbe** | [PingProbe\_API-Response-Time-Tester](https://github.com/Lakshit-Chiranjiv/Generic-Mini-Projects/tree/main/PingProbe_API-Response-Time-Tester) | API benchmarking tool that pings any public endpoint multiple times, streams live latency readings, shows avg/min/max/p95 stats, a color-coded Chart.js timeline, and a full per-request log with status badges. | — |
